1. Viewは何をしているか

Controllerの指示によって アクセスしてきたユーザーのブラウザに表示するデータを生成する。PHP/Laravelのカリキュラムの序盤で扱った、htmlファイルを出力するところ

2. プログラマーがHTMLを書かずにPHPなどのプログラミング言語やフレームワークを使う必要がある理由

作業分担ができること

3.

public function add() { return view('admin.profile.create'); } resources/views/admin/profileにcreate.blade.php public function edit() { return view('admin.profile.edit'); } resources/views/admin/profileにedit.blade.php
